Podjąłem decyzję – na blogu pojawią się reklamy!<!--more-->

<ad-></ad->

Postanowiłem, ze nadeszła pora, by zmonetyzować swoją wiedzę. Przez wszystkie te lata dzieliłem się nią za darmo, ale być może w końcu nadszedł czas, by to zmienić.

<ad-></ad->

Zadbałem o to, by reklamy nie były zbyt agresywne. Dzięki temu są dopasowane do treści i nie przeszkadzają za bardzo przy zapoznawaniu się z nią.

<ad-></ad->

<ad-></ad->

Również bloki kodu doczekały się własnych reklam:

<code-ad>

```typescript
const isNumber = (value: unknown): value is number => typeof value === 'number'
const isEven = (value: unknown): boolean => isNumber(value) && value % 2 === 0
const isOdd = (value: unknown): boolean => isNumber(value) && !isEven(value)
```



</code-ad>

Mam nadzieję, że ta mała, nieinwazyjna zmiana się spodoba!
